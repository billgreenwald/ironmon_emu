# IronMon Emu — Ironmon Tracker for Android

A GBA emulator for Android with the full [Ironmon Tracker](https://github.com/besteon/Ironmon-Tracker) built in — no PC, no setup, no second screen required.

**[Download the latest APK →](https://github.com/billgreenwald/ironmon_emu/releases/latest)**

## Features
- Live tracker panel: your Pokémon's stats, nature, ability, moves, type matchups
- Opponent tab: enemy species, level, revealed moves as the battle unfolds
- Routes tab: current location, wild encounters, trainer defeat counts
- GachaMon star rating with ruleset selector (Standard, Kaizo, Ultimate, etc.)
- Batch ROM families + UPR-Android integration for auto-randomizing runs
- Gamepad support with auto-hide on-screen controls
- Route map and hidden item image galleries (FR/LG)

## Requirements
- Android 8.0+
- A GBA ROM: Fire Red, Leaf Green, Ruby, Sapphire, or Emerald

## Install
Download `ironmon_emulator.apk` from the [Releases](https://github.com/billgreenwald/ironmon_emu/releases) page and sideload it. See the [README](https://github.com/billgreenwald/ironmon_emu) for full install steps.
